CSV Import/Export Guide

📋 How It Works

📌 First Step: Click "Export CSV" to download a template file with the correct format.

1

Export Template

Click Export CSV to download a file with the correct column headers. Open this file in Excel, Google Sheets, or any spreadsheet application.

2

Add Your Trades

Enter your trade data in the spreadsheet. The columns must remain in this exact order:

Type,Date,Symbol,Direction,Size,Entry,Exit,Stop Loss,Take Profit,P/L,Notes

Example:

Closed,2024-01-15,BTCUSDT,long,1,42000,43500,41500,44000,1500,"Great trade"
Active,2024-01-16,ETHUSDT,long,2,2200,,2150,2300,,"Still open"
3

Import Back

After editing, click Import CSV and select your file. Choose whether to replace existing data or append to it.

💡 Important Tips

  • Always export first to get the correct column format
  • For active trades, leave Exit and P/L columns empty
  • For closed trades, fill in both Exit and P/L
  • Use Replace mode to overwrite all existing data
  • Use Append mode to add new trades to your existing data
  • Keep a backup of your CSV files for safety
